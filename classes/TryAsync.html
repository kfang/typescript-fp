  • all<B>(arr: TryAsync<B>[]): TryAsync<B[]>
    • Defined in lib/try-async.ts:44

    Takes an array of TryAsync and returns a single TryAsync containing all the inner values as B[] if they are all successful.

    example

    TryAsync.all([TryAsync.success(1), TryAsync.success(2), TryAsync.success(3)]) => TryAsync.success([1, 2, 3])

    example

    TryAsync.all([TryAsync.success(1), TryAsync.failure(error), TryAsync.success(3)] => TryAsync.failure(error)
